# Bookmark Manager 
A browser-based Bookmark Manager built using HTML, CSS, and JavaScript as part of Round 1 of the University Internal Hackathon.  
Developed collaboratively in a team of four, this project focuses on efficient bookmark organization with a clean, responsive UI.

## Features
-  Add, edit, and delete bookmarks effortlessly  
-  Search and filter bookmarks by title or tag  
-  Persistent storage using Browser APIs  
-  Responsive and minimalistic user interface  

## Technologies Used
- HTML5 – Structure and layout  
- CSS – Styling and responsiveness  
- JavaScript – Functionality and interactivity  
- Browser APIs – Data storage and management
